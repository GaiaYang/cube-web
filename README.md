# 方塊資源網站

個人前端教學與作品展示網站，整理學習內容、實作範例與專案經驗，
可作為技術紀錄、作品集與對外履歷展示使用。

[🔗 Live Demo](https://void-cube.vercel.app)

## 專案目標

- 建立可長期維護的個人教學與資源整理平台
- 以實際專案呈現前端開發能力與設計思維
- 讓瀏覽者能快速理解技術取向與實作方式

## 使用技術

- 使用 Next.js（React + TypeScript） 建立模組化網站架構
- 採用 Tailwind CSS + DaisyUI 實作一致且可維護的 UI 樣式
- 自製 SVG 組合元件，用於動態樣式與視覺控制
- 使用 React Hook Form + Zod 實作表單與資料驗證
- 以 Jotai 進行狀態管理，降低元件耦合度
- 使用 Jest 進行核心功能測試，確保行為正確性
- 支援 深色模式與 RWD（手機 / 平板 / 桌機）
- 部署於 Vercel，搭配 GitHub 進行版本控管與部署流程

## 功能簡介

- 教學文章與技術內容整理
- 專案與實作範例展示
- 多裝置自適應版型（RWD）
- 深色 / 淺色主題切換
